# Yapper iOS Shutdown Notice

This folder contains a small static site (docs/) that explains the planned shutdown/rebranding notice for the iOS version of Yapper.

How to publish

- GitHub Pages can serve the content from the `docs/` folder on the repository's `main` branch if Pages is enabled.
- If you want me to open a Pull Request merging the `start-overlay-splash` branch into `main` and enable Pages, tell me and I will open the PR.

Notes

- The page currently references the repository's splash image via the raw.githubusercontent.com URL. You can copy the image into `docs/assets/` if you prefer local assets.
- Review the wording on the page for legal or trademark concerns before making it public.
